## Spotifou - Complete Learning Scope

### Core Features

| Feature | Description |
|---------|-------------|
| Music Catalog | Browse artists, albums, tracks |
| Playback | Play/pause, next/previous, progress bar |
| Search | Find songs, artists, albums |
| Playlists | Create, edit, add/remove tracks |
| User Auth | Login, register, profile |
| Library | Save favorites, liked songs |

### Frontend (React) Learning Focus

| Area | Tools/Concepts |
|------|----------------|
| Component architecture | Composition, props, custom hooks |
| State management | Context / Redux / Zustand |
| Routing | React Router, protected routes |
| Forms | React Hook Form + Zod validation |
| Data Fetching | TanStack Query |
| Performance | useMemo, useCallback, lazy loading |
| Animations | Framer Motion |
| Testing | Vitest + React Testing Library |
| UI/Styling | Tailwind or Material UI |

| Area | Concepts |

| Database | JPA/Hibernate, entities, repositories |
| Security | Spring Security, JWT authentication |
| File/Audio | Streaming, file upload/storage |
| Service Layer | Business logic, transactions |
| Validation | Bean validation, error handling |
| Testing | JUnit, MockMvc, integration tests |
| Caching | Spring Cache / Redis |
| Documentation | OpenAPI/Swagger |
